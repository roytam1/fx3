Customize the Mozilla Browser - Part Three

This screen enables you to customize the default home page, the text that appears in the title bar of the browser window, and the links that appear in the Personal Toolbar.

Click the Show Example buttons to see examples of the corresponding customizations. 

Default Home Page

Home Page URL. Enter the complete URL for the page you want your users to see when they start the Mozilla browser or click the Home button. 

Browser Window's Title Bar Text

Custom Title Bar Text. Enter the name you want to appear after the page title in the browser's title bar; for example, if you type "Mach Networks", " - Mach Networks" appears in the title bar after the name of the current page.

Custom Personal Toolbar

This section allows you to specify the names and corresponding URLs of the buttons that appear in the browser's Personal Toolbar.

Feedback. Click this button to go to a web page where you can provide feedback about the CCK Tool.

Next. Each time you click Next in a CCK screen, the information you entered or changed on that screen gets automatically saved with the selected configuration.

Exit. Click this button to exit the CCK Tool. The customizations you have specified on any preceding screens will be saved, but information you have specified on the current screen will not.
